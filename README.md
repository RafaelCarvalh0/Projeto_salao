# 📌 Projeto Salão

> Landing page responsiva para apresentação de serviços de um salão de beleza, com foco em exibição de trabalhos e presença digital.

---

## 🧠 Contexto

Este projeto foi desenvolvido no início dos meus estudos, com base em uma necessidade real.

A proposta surgiu a partir da rotina de atendimento de um salão de beleza, onde o agendamento era feito exclusivamente via WhatsApp, dificultando o gerenciamento e o retorno rápido às clientes.

Como solução inicial, foi criada uma página web para exibir os trabalhos realizados, com o objetivo de melhorar a apresentação dos serviços e, futuramente, facilitar o contato e agendamento.

> ⚠️ Projeto desenvolvido como estudo, com foco em estrutura e responsividade.
